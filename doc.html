競賽章程

國立臺南大學第31屆『南大盃』排球邀請賽

一、宗旨 : 提倡大專校院排球運動風氣，促進校際運動交流與發展。
二、指導單位 : 中華民國大專院校體育總會。
三、主辦單位 : 國立臺南大學。
四、承辦單位 : 國立臺南大學體育室、國立臺南大學排球隊、國立臺南大學排球社。
五、協辦單位 : 臺南市體育總會排球委員會。
六、比賽日期 : 中華民國111年1月29、30日（星期六、日）。
七、比賽地點 : 國立臺南大學中山體育館及室外排球場
八、比賽組別 :

（一）大專男子組。
（二）大專女子組。
（三）校友男子組。
（四）校友女子組。

九、參加資格：凡中華民國大專院校在籍學生、畢業校友，均得依下列辦法組隊參加。

（一）大專男子組：

1.凡同一學校之在籍學生均可自由組隊。
2.106～110學年度大專排球聯賽公開一級之註冊球員不得報名參賽。
3.為維護參賽權益，同一學校只得報名一隊。

（二）大專女子組：

1.凡同一學校之在籍學生均可自由組隊。
2.106～110學年度大專排球聯賽公開一級之註冊球員不得報名參賽。
3.為維護參賽權益，同一學校只得報名一隊。

（三）校友男子組：

1.凡同一學校之畢業校友均可自由組隊，若該校有併校或更改校名之情事，經證實為同一學校，仍具參賽資格。
2.106～110學年度大專排球聯賽公開一級之註冊球員一隊至多得報名兩名。
3.曾為國家代表隊之註冊選手不得報名參賽。
4.為維護各校參賽權益，校友組同一學校錄取隊伍數上限為兩隊，南大木鐸校友則不包含在此限制。

（四）校友女子組：

1.凡同一學校之畢業校友均可自由組隊，若該校有併校或更改校名之情事，經證實為同一學校，仍具參賽資格。
2.106～110學年度大專排球聯賽公開一級之註冊球員一隊至多得報名兩名。
3.曾為國家代表隊之註冊選手不得報名參賽。
4.為維護各校參賽權益，校友組同一學校錄取隊伍數上限為兩隊，南大木鐸校友則不包含在此限制。

（五）每1位球員於各組只限報名1隊，若重覆報名則只能代表相對較先報名(以線上完成報名記錄時間為準)之球隊出賽。

（六）重覆報名選手若確定更改代表參賽隊伍，須選手本人知會大會，請為其報名球隊之教練於110年10月22日前告知大會更改，否則無效；重覆報名選手若未於規定時間內完成更改程序，大會有權依規章刪除該選手在其他隊的報名，不得異議。

十、報名辦法：

（一）採用此網站報名，請點擊以下網址報名

（二）隊名務必取中文名稱，否則不受理報名。
（三）如未在報名截止時間前填寫完整報名資訊則不得更改，不符合報名所需資料者則不受理報名。
（四）報名時間：自111年1月8日00:00時起至111年1月11日23：59時止，逾期恕不受理報名（以線上完成報名記錄時間為準）。
（五）報名費：

1.每一參加球隊繳交報名費新臺幣3,500元整（含500元保證金）。
2.報名費繳交：請於111年1月15日23：59前匯款至指定郵局帳戶，未繳交報名費之球隊不受理報名。
戶名：國立臺南大學排球社蔡昀臻
帳號：00311580457959

（六）經名單公布，確定為正取之隊伍請於進行匯款後，主動聯繫主辦並提供帳號末5碼以及匯款人姓名，方便主辦核對匯款資料，確保雙方權益。如將匯款資料留於盃賽網網頁上，也請聯繫主辦，以防資料確認上的缺漏。
（七）報名人數：

領隊、助理教練、教練及管理報名各1人，各隊球員（含隊長及自由球員0~2人）得報名12人。

（八）報名隊數：

1.大專男子組：上限12隊（優先順序以線上完成報名記錄時間為準）
2.大專女子組：上限09隊（優先順序以線上完成報名記錄時間為準）
3.校友男子組: 上限15隊（優先順序以線上完成報名記錄時間為準）
4.校友女子組：上限12隊（優先順序以線上完成報名記錄時間為準）

十一、比賽制度 :

（一）賽制由大會依各組參加隊數之多寡決定，若該組報名未滿3隊，則該組取消比賽。
（二）各組比賽均採三局二勝制，任一方先獲得25分（決勝局15分)，並且領先另一方至少2分為獲勝。
（三）大專組預賽採循環三取二制；校友組預賽採全數晉級。
（四）決賽籤位以預賽名次為依據決定之。
